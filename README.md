# toy-projects

간단한 토이 프로젝트 모음 저장소입니다.

## 설명

이 저장소에는 개인 학습용으로 만든 작은 프로젝트들이 들어갑니다.

## 구조

- `fraud-detection/` - 예시 프로젝트 중 하나

## 시작하기

1. 리포지토리를 클론합니다:

```bash
git clone https://github.com/kyjwise7-hub/toy-projects.git
cd toy-projects
```

2. 로컬에서 작업 후 커밋하고 푸시하세요:

```bash
git add .
git commit -m "chore: update"
git push
```

## 라이선스

별도 표기가 없으면 개인용으로 사용합니다.
